# PaperClip Community Agents

A library of shareable Paperclip agents — each one a focused specialist you can
drop into your own Paperclip organization with a single command.

These agents are distributed as a portable `paperclip/v1` company package. You
normally do **not** import the whole company — instead use `./add-agent.sh` to
add just the agent(s) you want, along with the upstream skills each one needs.

See `README.md` for the install contract and the per-agent catalog.
